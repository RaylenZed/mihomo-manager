# mihomo-manager

> ⚠️ **此项目已并入 [Fluxo](https://github.com/RaylenZed/fluxo)**

`mihomo-manager` 的所有功能已整合为 `fluxo-cli`，作为 Fluxo 项目的一部分继续维护。

## 迁移指引

请使用 Fluxo 一键安装（自动包含 `fluxo-cli`）：

```bash
curl -fsSL fluxo.click/install.sh | sudo bash
```

安装完成后使用 `fluxo-cli` 代替原 `mm` 命令：

```bash
fluxo-cli          # 进入交互菜单（原 mm）
fluxo-cli status   # 查看状态
fluxo-cli test     # 网络连通性测试
```
